Provides classes to make a chart request.
Contains classes that store information about the request, wrapping classes of this information and a mediator.

There are filter classes, which is a normal filter, the time filter and a Measure. There are to classes which contain all the information, DimChart and TowDimChart. CharHostBuilder helps to build DimChart and TwoDimChart. The ChartMediator mediates between tasks coming in for this package and things needed from other packages.
